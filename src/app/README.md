# `src/app/` 디렉토리 요약

이 디렉토리는 런타임 앱 계층의 경계입니다.

## 구성

- `legacy-main.js`
  - 현재 공개 런타임의 실제 진입점입니다.
  - 모듈식 `src/app/main/` 코드를 반영한 레거시 미러 역할도 함께 수행합니다.
  - 스냅샷/저장본 직렬화 계약 때문에 완전히 제거되지 않았습니다.
- `main/`
  - 유지보수의 중심이 되는 모듈식 구현 디렉토리입니다.
  - 기능별 파일이 분리되어 있어 저장/병합/UI/데이터 흐름을 추적하기 좋습니다.

## 왜 `legacy-main.js`가 남아 있나

- 저장된 HTML은 직렬화된 런타임 조각을 다시 열어야 합니다.
- 이 과정에서 레거시 미러 경로가 아직 계약 검증 대상입니다.
- 따라서 이름은 legacy지만, 단순한 폐기 파일이 아니라 호환성 유지 경계입니다.

## 수정 우선순위

1. 새 기능/버그 수정은 우선 `main/`에서 검토
2. 저장본/재오픈 계약에 영향이 있으면 `legacy-main.js` parity 확인
3. `scripts/verify_snapshot_contract.js`로 미러 계약 검증
