# 키보드 내비게이션 빠른 참조 가이드

## 🎹 키보드 단축키

### 전역 단축키
| 키 | 설명 |
|----|------|
| `Tab` | 다음 요소로 포커스 이동 |
| `Shift+Tab` | 이전 요소로 포커스 이동 |
| `Enter` | 현재 포커스 요소 클릭/활성화 |
| `Space` | 현재 포커스 요소 클릭/활성화 (버튼용) |
| `Escape` | 드롭다운/모달 닫기 |

### 탭 전환 (사이트별 모드)
| 키 | 설명 |
|----|------|
| `→` | 다음 탭으로 이동 |
| `←` | 이전 탭으로 이동 |
| `Enter` | 현재 탭 선택 |

### 모드 전환
| 키 | 설명 |
|----|------|
| `→` | 다음 모드로 이동 (전체현황 → 사이트별) |
| `←` | 이전 모드로 이동 (사이트별 → 전체현황) |
| `Enter` | 현재 모드 선택 |

### 콤보 박스 (사이트 선택기)
| 키 | 설명 |
|----|------|
| `Enter`/`Space` | 드롭다운 열기/닫기 |
| `↓` | 다음 사이트로 이동 |
| `↑` | 이전 사이트로 이동 |
| `Enter` | 현재 사이트 선택 |
| `Escape` | 드롭다운 닫기 |

### 사이트 카드 (전체현황 모드)
| 키 | 설명 |
|----|------|
| `↓` | 다음 사이트 카드로 이동 |
| `↑` | 이전 사이트 카드로 이동 |
| `Enter` | 현재 사이트 상세 보기 |

---

## 🎨 포커스 표시

모든 요소는 포커스 시 다음과 같이 표시됩니다:
- **하늘색 테두리** (2px solid #0ea5e9)
- **빛나는 효과** (box-shadow)
- **닫기 버튼만** 빨간색 포커스 (#ef4444)

## 📋 탭 순서

### 기본 순서
1. 닫기 버튼 (X)
2. 저장 버튼
3. 새로고침 버튼
4. 전체현황 모드 버튼
5. 사이트별 모드 버튼
6. 사이트 선택기 콤보 버튼 (사이트별 모드時)
7. 탭 버튼들 (개요, 일별, URL, 검색어, 색인, 크롤, 백링크, 패턴, 인사이트)
8. 사이트 카드들 (전체현황 모드時)
9. 콘텐츠 영역 (탭 패널)

## 🎯 사용 예시

### 예시 1: 사이트별 보기로 전환하여 탭 탐색
```
1. Tab 키로 "사이트별" 버튼 포커스
2. Enter 키로 사이트별 모드 활성화
3. Tab 키로 "개요" 탭 포커스
4. → 키로 "일별" 탭 이동
5. → 키로 "URL" 탭 이동
6. Enter 키로 "URL" 탭 활성화
```

### 예시 2: 전체현황에서 사이트 카드 탐색
```
1. Tab 키로 첫 번째 사이트 카드 포커스
2. ↓ 키로 다음 사이트 카드 이동
3. ↓ 키로 또 다른 사이트 카드 이동
4. Enter 키로 선택한 사이트 상세 보기
```

### 예시 3: 콤보 박스로 사이트 변경
```
1. Tab 키로 사이트 선택기 포커스
2. Enter 키로 드롭다운 열기
3. ↓ 키로 원하는 사이트 탐색
4. Enter 키로 사이트 선택
```

## 💡 접근성 팁

1. **명확한 포커스**: 모든 인터랙티브 요소가 파란색 테두리로 표시됩니다
2. **논리적 순서**: Tab 키는 화면 상단→하단, 좌→우 순서로 이동합니다
3. **빠른 탐색**: 방향키를 사용하면 목록 내에서 빠르게 이동할 수 있습니다
4. **언제든 취소**: Escape 키로 드롭다운을 즉시 닫을 수 있습니다

## 🔧 개발자 참고

### 추가된 ARIA 속성
- `role="button"`, `role="tab"`, `role="tablist"`, `role="tabpanel"`
- `role="combobox"`, `role="option"`
- `aria-selected`, `aria-expanded`, `aria-controls`, `aria-label`

### 추가된 tabindex
- 모든 인터랙티브 요소에 `tabindex="0"` 추가
- 논리적인 탭 순서 보장

### 포커스 관리
- 포커스가 드롭다운 내부에 있을 때 ESC 키 누르면 트리거 버튼으로 포커스 복귀
- 방향키 이동 시 자동으로 포커스 이동
- 사이트 카드 클릭 후 사이트별 뷰로 전환
