# UI Runtime Update — 2026-03-19

## 업데이트 요약

이번 업데이트에서는 SearchAdvisor 패널의 UI 일관성, 캐시 메타 표현, 콤보박스 표시, 자동 새로고침 동작을 집중 정리했습니다.

## 반영 항목

- empty-state SVG 아이콘 렌더 문제 수정
- warm theme 블랙/옐로우/오렌지 톤 통일 강화
- 상태 UI / snapshot 저장 HTML 톤 보정
- 디자인 토큰(`T`) 1차/2차 도입
- 콤보박스 항목 제목을 실제 도메인 기준으로 표시
- live data가 있는 경우 부팅 직후 과도한 auto refresh 완화
- 캐시 메타 영역을 compact summary 1칩으로 축약

## 사용자 체감 변화

### 이전
- 콤보 항목 이름이 `https:`로 보일 수 있었음
- 캐시 정보가 헤더에서 공간을 과하게 차지함
- 일부 환경에서 캐시를 무시하는 것처럼 자동 새로고침이 동작할 수 있었음

### 이후
- 콤보 항목 제목은 `도메인`, 보조 줄은 `전체 URL`
- 캐시 정보는 compact summary 1칩으로 표시
- 페이지 안에 fresh data가 있으면 부팅 직후 자동 재수집이 덜 공격적으로 동작
