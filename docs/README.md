# 코드 디렉토리 문서 인덱스

이 폴더는 공개 저장소 기준의 최소 문서 진입점입니다.

아래 링크는 실제 코드가 위치한 디렉토리의 요약 문서로 연결됩니다.

- [루트 개요](../README.md)
- [`src/` 런타임 엔트리와 상위 레이어](../src/README.md)
- [`src/app/` 레거시 진입점과 모듈식 앱 경계](../src/app/README.md)
- [`src/app/main/` 핵심 런타임 모듈 상세](../src/app/main/README.md)
- [`scripts/` 빌드 검증/배포 보조 스크립트](../scripts/README.md)
- [`dist/` 공개 배포 산출물](../dist/README.md)
- [`.github/` CI 워크플로우](../.github/README.md)

## 읽는 순서 추천

1. `README.md`
2. `src/README.md`
3. `src/app/README.md`
4. `src/app/main/README.md`
5. 필요할 때 `scripts/README.md`, `dist/README.md`

## 문서 원칙

- 코드 자체는 설명하지, 재작성하지 않습니다.
- 기능 계약은 실제 소스가 우선입니다.
- 저장/병합/엑셀/스냅샷 흐름은 `src/app/main/` 문서를 기준으로 따라가면 됩니다.
