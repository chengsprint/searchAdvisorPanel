## 🔴 Critical 문제 (즉시 수정 필요)

### 1. innerHTML XSS 취약점
- **위치**: 02-dom-init.js, 08-renderers.js, 10-all-sites-view.js, 11-site-view.js, 12-snapshot.js, 13-refresh.js, 14-init.js
- **문제**: 사용자 입력이 포함될 수 있는 데이터에 innerHTML 직접 사용
- **해결**: innerHTML → textContent 또는 DOM API로 변경

### 2. 함수 중복 정의 (순환 참조)
- **위치**: buildSiteSummaryRow()가 10-all-sites-view.js와 11-site-view.js에 중복 정의
- **문제**: 순환 참조 위험, 유지보수 어려움
- **해결**: 단일 파일(01-helpers.js)로 통합

### 3. 거대 함수
- **위치**:
  - 08-renderers.js: buildRenderers() (720줄)
  - 12-snapshot.js: buildSnapshotHtml() (622줄)
- **문제**: 유지보수 어려움, 테스트 불가
- **해결**: 함수 분리
